## 这是什么？

**MoonCake Studio** 是一个完全运行在浏览器端的 3D 参数化设计工具，让你自由定制月饼模具——那种中秋节用来压印月饼的传统模具。

**零后端、零安装、零构建**。打开浏览器就能用。

你可以定制边缘轮廓、表面花纹、刻字、配色，然后导出生产级 **STL** 或 **GLB** 文件，直接送去 3D 打印或 CNC 加工。

## 功能亮点

### 🎨 设计系统
- **6 种边缘轮廓** — 光滑、波浪、花瓣、梅花、菊花、锯齿
- **6 种传统花纹** — 花卉纹、祥云纹、五谷纹、缠枝纹、窗棂纹、海浪纹
- **AI 随机花纹** — 程序化生成曼陀罗 / 莲花 / 几何星纹，无限变化
- **自定义图片转浮雕** — 上传任意图片，自动转换为浮雕纹理
- **中文刻字** — 两字（左右）或四字（田字）布局，5 种字体可选

### 🔧 参数化控制
| 参数 | 范围 | 说明 |
|------|------|------|
| 直径 | 6–12 cm | 模具外径 |
| 总高度 | 1.5–4 cm | 模具总体高度 |
| 顶面弧度 | 0–1.5 | 月饼顶面的穹顶弧度 |
| 花边深度 | 0–1.5 | 边缘波浪的振幅 |
| 花边瓣数 | 0–32 | 边缘花瓣的数量 |
| 文字大小 | 0.4–1.5× | 刻字缩放比例 |
| 雕刻深度 | 0.05–0.8 cm | 花纹和文字的雕刻深度 |

### 🏭 双视图模式
- **成品预览** — 查看压制完成的月饼成品效果
- **模具预览** — 完整的工业模具装配体：外壳、导杆、弹簧、印模板、压柄

### 📦 导出与打印
- **STL 导出** — FDM / SLA 3D 打印和 CNC 通用格式
- **GLB 导出** — glTF 二进制，适用于游戏引擎、AR/VR、Web 查看器
- **打印分析** — 实时显示尺寸(mm)、三角面数、PLA 估重、打印耗时
- **FDM 优化** — 推荐层高 0.15mm，填充率 20%

### 🖥️ 专业级 UI
- 暗色主题，参考专业 CAD 工具设计
- 可折叠参数面板 + 实时数值反馈
- 尺寸标注覆盖层
- 线框检查模式
- 轨道 / 平移 / 缩放控制 + 双击重置
- 桌面端和平板端响应式布局

## 使用指南

### 基本流程

1. **选择边缘样式** — 在「模具外形」区域点选边缘轮廓
2. **选择花纹** — 选择传统纹样或点击「AI 随机生成图案」
3. **添加文字** — 输入最多 4 个汉字，选择字体和布局
4. **调整参数** — 用滑块微调直径、高度、弧度、雕刻深度
5. **选择配色** — 从 6 套精选配色方案中选择
6. **双视图预览** — 在成品预览和模具预览之间切换
7. **导出** — 下载 STL 用于 3D 打印，或 GLB 用于数字展示

### 自定义图片花纹

1. 点击上传区域或拖拽图片
2. 图片自动转为高度图浮雕
3. 调节「背景清除范围」滑块控制中心遮罩
4. 文字刻印叠加在上方

### 3D 打印建议

- **材料**: 推荐 PLA，食品接触场景用 PETG
- **层高**: 0.15mm，确保花纹细节
- **填充率**: 20% 即可满足结构强度
- **方向**: 模腔朝上打印，无需支撑
- **后处理**: 轻度打磨，保证食品接触面光滑

## 技术架构

| 技术 | 用途 |
|------|------|
| **Three.js r128** | 3D 渲染、几何体生成、阴影映射 |
| **原生 JavaScript** | 零框架、零构建步骤 |
| **Canvas 2D API** | 程序化花纹绘制、高度图生成 |
| **LatheGeometry + 位移映射** | 参数化模具本体 + 实时顶点位移 |
| **STLExporter / GLTFExporter** | 生产级 3D 模型导出 |
| **CSS 自定义属性** | 可主题化暗色 UI + 响应式布局 |

### 核心技术亮点

- **单文件部署** — 整个应用是一个自包含的 HTML 文件（约 1300 行）
- **实时参数化建模** — 每次参数变更都使用 LatheGeometry + 顶点级位移映射重建几何体
- **程序化花纹引擎** — 6 种传统纹样 + AI 生成纹样，全部通过 Canvas 2D 贝塞尔曲线绘制，转为高度图驱动 3D 位移
- **双用途渲染** — 同一组参数数据同时驱动月饼成品预览和完整工业模具装配体（外壳、导杆、弹簧、压柄）
- **双线性高度采样** — 高度图以亚像素双线性插值采样，确保位移平滑
